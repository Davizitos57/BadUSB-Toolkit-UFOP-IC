# DigiSpark Toolkit — UFOP/IC  
Repositório de scripts, experimentos e documentação técnica desenvolvidos no projeto de Iniciação Científica da Universidade Federal de Ouro Preto (UFOP), no contexto do edital 06/2025 – PIVIC-2S/UFOP-2025-2026.

Este repositório reúne códigos e análises voltados à simulação de ataques físicos utilizando dispositivos BadUSB — especialmente **DigiSpark** e **Rubber Ducky** — com foco em demonstração de riscos, avaliação de impacto e estudo de estratégias de mitigação.

---

## 📌 Objetivo do Projeto
O objetivo central deste projeto é **investigar, simular e documentar ataques BadUSB**, avaliando seu impacto e propondo mecanismos eficazes de defesa.

Dispositivos como o DigiSpark (Arduino ATTiny85) e o Rubber Ducky (HID Scriptable Keyboard) podem executar comandos automaticamente ao serem conectados a um computador, tornando-se ferramentas reais para ataques silenciosos e de rápida execução.

Este repositório complementa o estudo acadêmico fornecendo **scripts, evidências, testes, documentação e boas práticas de mitigação**.
